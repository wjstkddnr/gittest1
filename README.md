#  프로젝트

## 👨‍🏫 프로젝트 개요
  
  ## :alarm_clock: 문제인식
  #### 우리나라의 주식투자자 수는 계속 증가하고 있음에도 우리나라의 주식 투자 비율은 2~30% 정도로 7~80%의 미국에 비해 매우 작은 상태입니다. 하지만 국내 개인투자자 대부분은 바쁜 일상에 치여 제대로된 증권 정보를 얻기가 쉽지 않은 실정입니다. 증가하고 있는 주식 투자자들의 효과적으로 정보를 얻기 위한 툴에 대한 수요는 앞으로도 증가할 것으로 판단되며 이로 인해 우리는 주식 투자자들에게 효과적으로 실시간 증권 시황과 뉴스 정보를 제공하는 챗봇을 구현하려 합니다.


## 👩‍🏫 서비스 목표
  #### 바쁘디 바쁜 현대 사회를 살아가는 가운데 편리하고 빠르게 주식에 대한 정보를 손쉽게 알게 되기 위해 이 LLM 모델을 구성하였습니다.
  #### 서비스는 10분마다 업데이트를 하여 최신의 정보를 사용자에게 제공해 주려고 합나다.
